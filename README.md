# ⭐ Family Activity Tracker

A real-time chore tracking and rewards app for the family. Sophia and Olivia earn points for completing daily tasks and redeem them for rewards!

## Features
- 📋 Daily task tracking with categories (Cleanliness, Dogs, School, Health, Household)
- ⭐ Point system with configurable values
- 🎁 Reward shop where kids spend earned points
- 🏆 Scoreboard with streaks and weekly summaries
- 👑 Parent dashboard with activity logs and full control
- 🔒 PIN protection for parent accounts
- 📱 PWA — installable on phones as a native app
- ⚡ Real-time sync across all devices via WebSocket

## Running Locally
```bash
npm install
node server.js
```
Open `http://localhost:3000` on your phone or browser.

## Hosted on Render
This app auto-deploys from GitHub. Any push to `main` triggers a new deploy.
